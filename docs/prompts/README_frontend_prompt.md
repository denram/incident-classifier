# Prompt para gerar README.md do frontend

Você é um Frontend Architect e Technical Writer especializado em documentação de aplicações web. Analise toda a estrutura e código-fonte deste projeto frontend e gere um README.md profissional em Markdown.

Contexto do projeto:
O projeto se chama: "Sistema Inteligente de Classificação de Ocorrências - SICO"

"Sistema Inteligente de Classificação de Ocorrências"

Objetivo do frontend:
Permitir que operadores de portaria:

- descrevam ocorrências em linguagem natural
- enviem a ocorrência para processamento
- visualizem o retorno estruturado da IA

O sistema recebe do backend:

- texto formalizado
- categoria
- gravidade
- sugestão de ação

Tecnologias esperadas:

- Angular
- TypeScript
- HTML/CSS
- Consumo de API REST

Mencione que a IA foi utilizada como ferramenta de desenvolvimento (vibe coding/assistente de codificação), não como funcionalidade do sistema.

Sua tarefa:

1. Ler o projeto frontend completo
2. Identificar:
   - componentes
   - páginas
   - serviços
   - estrutura de pastas
   - integração com API
   - fluxo de navegação
3. Gerar um README.md claro, organizado e adequado para um mini projeto acadêmico.

O README deve conter:

- Título do projeto
- Descrição geral
- Objetivo da interface
- Papel da IA como ferramenta de desenvolvimento
- Explicação simples da arquitetura frontend
- Tecnologias utilizadas
- Estrutura de pastas
- Fluxo da aplicação
- Componentes principais
- Integração com backend
- Como executar localmente
- Exemplos visuais esperados
- Limitações conhecidas e melhorias futuras

IMPORTANTE:

- Este é um projeto acadêmico simples, focado em demonstrar:
  - integração funcional de IA
  - arquitetura desacoplada básica
  - boas práticas de desenvolvimento
  - documentação
- Não invente funcionalidades que não existam no código
- Não exagerar complexidade técnica
- O README deve parecer acadêmico, porém profissional
- Priorize clareza e simplicidade
- Retorne apenas o conteúdo final do README.md
