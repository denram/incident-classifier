# incident-classifier

Incident Classifier - Classificador de incidentes utilizando IA.

## Estrutura do Projeto

```
incident-classifier/
│
├── frontend/       # Aplicação frontend
│   └── README.md
│
├── backend/        # API REST Java + Spring Boot
│   ├── src/
│   ├── pom.xml
│   └── README.md
│
├── docs/           # Documentação do projeto
│   └── prompts.md
│
└── README.md
```

## Módulos

- **backend**: API REST desenvolvida em Java 25 + Maven + Spring Boot, responsável pela classificação de incidentes via IA.
- **frontend**: Interface web do classificador de incidentes.
